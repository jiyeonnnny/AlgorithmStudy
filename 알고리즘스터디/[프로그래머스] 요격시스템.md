# [프로그래머스] 요격시스템
```py
def solution(targets):
    answer = 0
    targets.sort(key=lambda x : x[0])

    e1 = targets[0][1]
    for i in range(len(targets)):
        if e1<=targets[i][0]:
            answer += 1
            e1 = targets[i][1]
        elif targets[i][0]<e1<=targets[i][1]:
            pass
        else:
            e1 = targets[i][1]
        if i==len(targets)-1:
            answer+=1
    return answer
```
1. 정렬 s 기준 오름차순
2. e1 가정
   - e1<=s  answer+=1 / new_e1=e
   - s<e1<=e  pass
   - e<e1  new_e1=e
