3. list
확장  --  list.extend()
추가  --  list.append()
```(python)
a = [1,2,3]
b = [4,5,6]
a.extend(b)
> a = [1,2,3,4,5,6]
a.append(b)
> a = [1,2,3,[4,5,6]]
```
