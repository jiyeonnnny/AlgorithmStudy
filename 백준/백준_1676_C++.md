# [백준 1676] C++ 문제 풀이
```cpp
#include<iostream>
#include<string>
using namespace std;
int N, num=1;

int main(void) {
	cin >> N;
	for (int i = 1; i <= N; i++) {
		num *= i;
	}
	string str = to_string(num);
	int len = str.length();
	int count=0;
	for (int i = len - 1; i >= 0; i--) {
		if (str[i] == '0') {
			count += 1;
		}
		else break;
	}
	cout << count << endl;
	return 0; 
}
```

- 파이썬이랑 동일한 코드 작성
